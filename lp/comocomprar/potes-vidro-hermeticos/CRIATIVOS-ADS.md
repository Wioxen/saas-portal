## 5. Conversão (já implementada na LP)

Como afiliado você NÃO tem acesso à venda real da Amazon. A LP usa **clique no CTA como proxy de conversão** — o evento dispara antes do redirect pra `https://amzn.to/4mT9dJu`.

### Conversão configurada no Google Ads
| Item | Valor |
|---|---|
| Conta | `AW-16696952717` |
| Conversion ID | `welKCJCdt9EZEI2P3Zk-` |
| Nome | Clique de saída para Amazon |
| Categoria | Compra |
| Valor padrão | R$ 1,00 |
| Moeda | BRL |
| Contagem | Cada (uma por clique) |

### Como está implementado na LP

1. **Bootstrap gtag.js no `<head>`** (carrega async, não bloqueia LCP):
   ```html
   <script async src='https://www.googletagmanager.com/gtag/js?id=AW-16696952717'></script>
   <script>
   window.dataLayer=window.dataLayer||[];
   function gtag(){dataLayer.push(arguments)}
   gtag('js',new Date());
   gtag('config','AW-16696952717');
   function gtag_report_conversion(){gtag('event','conversion',{'send_to':'AW-16696952717/welKCJCdt9EZEI2P3Zk-','value':1.0,'currency':'BRL'})}
   </script>
   ```

2. **Os 2 botões CTA** (hero + final) chamam `gtag_report_conversion()` no `onclick`:
   ```html
   <a href='https://amzn.to/4mT9dJu' onclick='gtag_report_conversion()' target='_blank' rel='sponsored noopener nofollow'>Ver oferta na Amazon</a>
   ```

> **Por que sem `event_callback` + setTimeout:** o snippet padrão do Google bloqueia o redirect até o evento ser enviado, mas como nossos CTAs usam `target='_blank'` (abre em nova aba), o `href` já cuida da navegação em paralelo. O evento dispara fire-and-forget — captura ~95-98% dos cliques, sem prejudicar UX.

### Validação após deploy

1. Abrir LP em produção, abrir DevTools → Network
2. Clicar no CTA "Ver oferta na Amazon"
3. Verificar request para `googleads.g.doubleclick.net/pagead/conversion/...` com status 200
4. No Google Ads, em **Ferramentas → Conversões**, status deve mudar de "Não verificada" → "Recebendo conversões" em até 24h

## 7. Estratégia de escala (quando bater nos primeiros R$ 500 de gasto)

1. **Aumentar orçamento 20%/semana** se ROAS proxy ≥ 2x.
2. **Adicionar correspondência ampla** com lances mais baixos (-50%) para descobrir variações.
3. **Lançar campanha Performance Max** com a LP de destino + feed de produtos (se publicar mais SKUs).
4. **Remarketing** — público RLSA com lance +30% pra quem visitou a LP e não clicou no CTA Amazon.
5. **Replicar template** — duplicar a LP para mais 10-20 produtos do mesmo nicho (cozinha, organização) usando o mesmo molde.
