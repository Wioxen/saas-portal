### 📋 Checklist Final para o Desenvolvimento (O "Briefing de Ouro")

> **"Claude, mãos à obra. Vamos construir o Core do SaaS PHP. Siga estas diretrizes técnicas finais:"**

#### 1. Gerenciamento de Falhas (Retry Logic)
* Implemente um sistema de **3 tentativas** para chamadas de API (Gemini e WordPress).
* Se o WordPress retornar erro, o post deve voltar para a fila com status `erro` e um log detalhado do motivo.

#### 2. Fluxo de Assets (Imagens)
* O script deve baixar a imagem base para uma pasta `/tmp`.
* O módulo **Imagick** processa e gera a versão `.webp`.
* Após o upload bem-sucedido para o WP, o script **deve apagar** os arquivos temporários para não lotar o disco da VPS de 2GB.

#### 3. Cabeçalhos de Segurança (Headers)
* Na conexão com o WordPress, use um `User-Agent` único (ex: `SaaS-Imperio-Alpha-1.0`).
* No lado do WordPress (functions.php), adicione a regra de bloqueio: *"Se a requisição for `/wp-json/` e o IP não for [SEU_IP], retorne 403"*.

#### 4. Otimização de Performance no PHP
* Use `PDO` com `prepared statements` para todas as consultas ao MySQL.
* Configure o script para registrar o tempo de execução de cada tarefa no log, para monitorarmos gargalos na VPS.

#### 5. Integração Google Indexing
* Certifique-se de que o script trata o retorno `200 OK` do Google e salva na tabela `posts_gerados` a coluna `indexado_em` (datetime).
