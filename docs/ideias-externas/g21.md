Este é um dos pilares mais importantes para o **Nível Extraordinário**, pois o Google Discover e o Google Imagens usam algoritmos visualmente agressivos para detectar conteúdo duplicado. Se você usa a mesma imagem do banco de imagens ou do site oficial que outros 1.000 sites usam, suas chances de viralizar caem drasticamente.

Para que sua aplicação PHP no Easypanel gere imagens **100% originais** aos olhos do Google, vamos usar a biblioteca **Imagick (ImageMagick)**, que é extremamente poderosa e superior à GD para manipulação profissional.

Aqui está o detalhamento técnico e a lógica expert para o seu **Módulo de Geração de Imagens Originais**:

---

### 1. A Lógica da "Originalidade Visual" (Branding Dinâmico)

Não vamos apenas redimensionar. O seu SaaS PHP deve criar uma **"Assinatura Visual"** única para cada post. O processo extraordinário funciona assim:

1.  **A IA gera o texto:** A IA (Gemini) gera o texto do post e sugere um "termo de busca" para a imagem (ex: *"Mulher estudando no computador com logo do SENAC desfoque"*).
2.  **Busca de Base:** O SaaS PHP busca uma imagem base (pode usar APIs como Pexels/Unsplash ou um banco próprio pré-carregado).
3.  **A Transformação (Imagick):** O PHP pega essa imagem base e aplica **camadas de modificação aleatória** para que o hash do arquivo mude completamente.

---

### 2. O Script PHP: Gerador de Imagens com Imagick

Este script demonstra como sua aplicação PHP pode pegar uma imagem base e transformá-la em um ativo único com marca d'água e filtros, pronto para o Discover.

```php
<?php

class GeradorImagemExtraordinario {

    private $logoPath;
    private $fontPath;

    public function __construct($logoPath, $fontPath) {
        $this->logoPath = $logoPath; // Path para o logo do seu site (transparente)
        $this->fontPath = $fontPath; // Path para uma fonte TTF (ex: Roboto Bold)
    }

    /**
     * Gera uma imagem única com filtros, branding e texto
     */
    public function criarImagemUnica($imagemBase, $textoDestaque, $outputPath) {
        if (!file_exists($imagemBase)) return false;

        // 1. Carrega a imagem base
        $image = new Imagick($imagemBase);

        // 2. Otimização Base: Redimensiona para o padrão Discover (1200x675 ou 16:9)
        $image->cropThumbnailImage(1200, 675);
        $image->setImageFormat('webp'); // Discover ama WebP
        $image->setImageCompressionQuality(85);

        // --- CAMADA 1: Filtros de Originalidade ---
        // Aplica um leve ajuste de cor aleatório para quebrar o padrão de cores original
        // Isso muda o hash da imagem para o Google
        $hue = rand(-2, 2); // Leve variação de matiz
        $saturation = rand(100, 110); // Leve aumento de saturação
        $brightness = rand(95, 105); // Leve variação de brilho
        $image->modulateImage($brightness, $saturation, $hue);

        // Opcional: Adiciona um ruído leve para originalidade extrema
        // $image->addNoiseImage(Imagick::NOISE_GAUSSIAN);

        // --- CAMADA 2: Branding (Sua Marca) ---
        // Adiciona o logo do seu site no canto (ex: vagasebeneficios.com.br)
        if (file_exists($this->logoPath)) {
            $logo = new Imagick($this->logoPath);
            $logo->evaluateImage(Imagick::EVALUATE_MULTIPLY, 0.7, Imagick::CHANNEL_ALPHA); // 70% de opacidade
            $logo->resizeImage(150, 0, Imagick::FILTER_LANCZOS, 1); // Redimensiona o logo

            // Posição do logo (inferior direito)
            $x = $image->getImageWidth() - $logo->getImageWidth() - 20;
            $y = $image->getImageHeight() - $logo->getImageHeight() - 20;

            $image->compositeImage($logo, Imagick::COMPOSITE_OVER, $x, $y);
            $logo->destroy();
        }

        // --- CAMADA 3: Texto Dinâmico (SEO) ---
        // Escreve o título ou palavra-chave na imagem (ex: "Inscrições Abertas!")
        if (!empty($this->fontPath)) {
            $draw = new ImagickDraw();
            $draw->setFont($this->fontPath);
            $draw->setFontSize(50);
            $draw->setFillColor('white');
            $draw->setGravity(Imagick::GRAVITY_NORTHWEST); // Texto no topo esquerdo

            // Adiciona uma sombra preta leve atrás do texto para leitura
            $draw->setTextAntialias(true);
            $draw->setStrokeColor(new ImagickPixel('black'));
            $draw->setStrokeWidth(2);
            $draw->annotation(52, 52, $textoDestaque); // Sombra

            $draw->setStrokeWidth(0);
            $draw->annotation(50, 50, $textoDestaque); // Texto branco principal

            $image->drawImage($draw);
            $draw->destroy();
        }

        // 3. Salva a imagem final no formato WebP
        $image->writeImage($outputPath);
        $image->destroy();

        return true;
    }
}

// --- Exemplo de Uso no seu SaaS PHP ---

// Configurações (Paths dentro do Easypanel)
$logoSite = '/var/www/html/assets/img/logo-vagas-branco.png';
$fonte    = '/var/www/html/assets/fonts/Roboto-Bold.ttf';
$imagemOriginal = '/var/www/html/temp/imagem-base-senac.jpg'; // Baixada temporariamente
$outputFinal = '/var/www/html/temp/post-senac-uníco.webp'; // Imagem pronta

$gerador = new GeradorImagemExtraordinario($logoSite, $fonte);

// Texto que a IA sugeriu para a imagem
$textoImagem = "Inscrições Abertas\nSENAC 2026!";

if ($gerador->criarImagemUnica($imagemOriginal, $textoImagem, $outputFinal)) {
    echo "🚀 Imagem extraordinária gerada com sucesso!";
    // Agora você faz o upload via API para o WordPress
} else {
    echo "❌ Erro ao gerar imagem.";
}
```

### 3. Integração com a API do WordPress

Após o PHP gerar a imagem única (`.webp`), o SaaS deve enviá-la para a biblioteca de mídia do site correto via API:

1.  **Endpoint:** `POST https://vagasebeneficios.com.br/wp-json/wp/v2/media`
2.  **Headers:** `Content-Disposition: attachment; filename="post-senac-unico.webp"`, `Content-Type: image/webp`.
3.  **Body:** Envie o conteúdo binário do arquivo gerado.
4.  **O SEO Final:** A API do WordPress retornará um ID. O seu SaaS PHP deve usar esse ID e enviar um `PUT` para atualizar o **Texto Alternativo (Alt Text)** da imagem com a palavra-chave do post (ex: *"Formulário de inscrição para cursos gratuitos do SENAC 2026"*).

---

### 4. Vantagens Extraordinárias desta Abordagem

1.  **Originalidade Técnica:** O Google analisa pixels e hashes. Seus filtros de matiz e ruído aleatórios quebram o padrão da imagem original, fazendo o Google achar que é uma foto nova.
2.  **Aprovação no Discover:** O Discover exige imagens com pelo menos 1200px de largura e alta qualidade. Esse script garante isso.
3.  **Monetização:** Imagens com CTAs visuais (ex: *"Clique para ver o calendário"*) aumentam a taxa de clique dentro do site, gerando mais impressões de AdSense.
4.  **Automatização Total:** Você não precisa de designer. Seu SaaS PHP vira uma fábrica de imagens.

### 5. Configuração no Easypanel (Docker)

Certifique-se de que a imagem Docker que você está usando para o seu SaaS PHP tenha o **ImageMagick** e a extensão PHP **Imagick** instaladas. Se não tiver, peça ao Claude para te ajudar a criar um `Dockerfile` customizado que inclua:

```dockerfile
# Exemplo de Dockerfile para PHP com Imagick
FROM php:8.2-fpm-alpine

# Instala dependências do ImageMagick
RUN apk add --no-cache \
    imagemagick \
    imagemagick-dev \
    libpng-dev \
    libjpeg-turbo-dev \
    libwebp-dev \
    freetype-dev

# Instala a extensão Imagick do PHP
RUN docker-php-ext-configure gd --with-freetype --with-jpeg --with-webp
RUN docker-php-ext-install gd pdo pdo_mysql mysqli
RUN pecl install imagick
RUN docker-php-ext-enable imagick

WORKDIR /var/www/html
```
